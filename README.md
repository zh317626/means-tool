
## 描述


### install

```
npm install means-tool
```

### introduce

```
import { files, means, regular } from 'means-tool/dist' // or
import means from 'means-tool/dist';
```

### employ

```
means.urlToObjet('https://********index?id=113565611goods_id=496455424019')
```
